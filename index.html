GITHUB CERTIFICATION

Nessa pasta irei reunir todo o conhecimento necessário para passar das provas da certificação do Git Hub!

A ordem para passar das certificações são (com nível de dificuldade):

  • Github Foundations (level 100)

  • Github Advanced Segurity (level 150) Obs : Recomendado para Cloud Segurity Engineer

  • Github Actions (level 200) Obs : Recomendado para Cloud Developer

  • Github Admin (level 300) Obs : Recomendado para Enterprise Engineer

Na prova terão 75 questões: 60 questões com score e outras 15 sem score (Obs: sendo assim, só poderá errar 15 questões e errar não te trará nenhuma penalidade), os formatos são de multipla escolha e multipla resposta.

E quando falo de 15 questões sem score, são questões que não contarão na nota final! Elas servem apenas para avaliar a introdução de novas perguntas, para determinar se o exame está fácil demais ou se a pontuação de corte / dificuldade precisa ser aumentada e para descobrir usuários que estão tentando trapacear ou roubar questões para "dumps" (banco de questões vazados)

Duração da prova : 2 horas (1.6 minutos per questão)
